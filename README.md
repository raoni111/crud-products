
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## </>Routes

GET, POST

- `http://localhost:3000/product`

PUT, DELETE

- `http://localhost:3000/product/:id`

GET BY CATEGORY

- `http://localhost:3000/product/by/category/:category`

GET BY ID

- `http://localhost:3000/product/by/id/:id`
